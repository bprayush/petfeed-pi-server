# PetFeed Pi Server

PetFeed Pi Server is a flask python server created for integration of web services 

### Prerequisites

1. Python3
2. pip3
3. flask
4. PyMysql
5. Rpi.GPIO
6. PusherClient
7. Pusher

### Installing

Install python3

Install Flask

```
pip install Flask
```

Install PyMySql
```
pip install pymysql
```

Install Pusher Client
```
pip install pusherclient
```

Install Puserh
```
pip install pusher
```
